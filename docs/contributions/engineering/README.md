# Engineering Contributions

If you have an idea on how to expand this extension so that it can further help engineers when using SketchUp, please 
voice it! Ideas like:

- An energy performance analysis tool that is integrated with the structures created by this extension.
- A stress calculation module to automatically determine the material and size of the structure components.
- A tool that can create piping arrangement or electrical cables.

All ideas are welcome! You can submit it via:

1. Open an issue in Github using this  [link](https://github.com/mustafkamal/sketchup-structureline/issues/new?template=contributions-in-engineering.md). Type the title, describe your idea, and click **Create**.
2. Reply to this [post](https://forums.sketchup.com/t/structure-line-early-release/345057) in the SketchUp Community page.
3. Reply to this [tweet](https://x.com/structureline31/status/2028015173404115092).
